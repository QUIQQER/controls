Produktverwaltung
========

Repository für alle standard Controls die QUIQQER mitliefert.
Controls sind PHP Klassen welche in HTML und somit GUI Element erzeugen.

Paketname:

    quiqqer/controls



Features (Funktionen)
--------

- QUI Hauptcontrol Klasse

Installation
------------

Der Paketname ist: quiqqer/controls
